# Grocery List App

## Introduction

An Android application that helps you keep track of the products you need while shopping

## Architecture

Pattern Model-View-ViewModel(MVVM) which facilitates a separation of development of the graphical user interface.

## Technologies used
* ViewModel 
* LiveData 
* Kotlin Coroutines 
* Room persistence
* Lottie
